# 客户

>

>

> {
>     "username":"admin",
>     "password":"ynuadmin"
> }
> ```

# 商品

## 查询商品

> 请求方式：Get
>
> 请求路径：/goods
>
> 请求参数：
>
> ``` txt
> Integer page,
> Integer pageSize,
> String goodsName, 
> String producer, 
> Double startPrice, 
> Double endPrice,
> Integer order,
> LocalDate startProduceTime,
> LocalDate endProduceTime
> ```

## 添加商品

> 请求方式：Post
>
> 请求路径：/goods
>
> 请求参数：json
>
> ```json
> {
>     "goodsName": "防晒霜",
>     "producer": "安耐晒",
>     "produceDate": "2023-07-02",
>     "type": 1,
>     "buyingPrice": 80,
>     "price": 100,
>     "inventory": 20
> }
> ```

## 通过id查询商品

> 请求方式:Get
>
> 请求路径:/goods/id
>
> 请求参数:id

## 修改商品

> 请求方式：PUT
>
> 请求路径: /goods
>
> 请求参数:json
>
> ```json
> {		
>    "goodsId": 1,//必须
>    "goodsName": "牙刷",
>    "producer": "洁丽雅",
>    "produceDate": "2022-01-01",
>    "type": 1,
>    "buyingPrice": 0.5,
>    "price": 1.5,
>    "inventory": 10
> }
> ```
>

## 删除商品

> 请求方式:Delete
>
> 请求路径:/goods/{ids}
>
> 请求参数:ids(路径参数)

# 购物车

## 向购物车添加商品

> 请求方式:Post
>
> 请求路径:/shoppingcart
>
> 请求参数:json
>
> ```json
> {
>     "userId":1,
>     "goodsId":1,
>    	"amount":2
> }
> ```

## 将商品移出购物车

> 请求方式:Delete
>
> 请求路径:/shoppingcart/{goodsIds}
>
> 请求参数:{goodsIds},json
>
> ```json
> {
>     "userId":3
> }
> ```
>
>
